### Документация к диалекту языка Rust ArrowRust

## Стрелки
ArrowRust использует стрелки как основной оператор языка. Схематично их можно представить как:
`аргументы -> функция() -> возврат`, но это не всё, что есть. Транслятор превращает это в валидный Rust-код, который работает ничуть не хуже нативного.

| Стрелка | Что делает | Пример |
|---------|------------|--------|
| `+>` | Передаёт несколько аргументов в функцию | `[аргумент1, аргумент2] +> функция()` |
| `->` | Передаёт значение в функцию или в переменную | `функция() -> переменная1 -> функция2()` |
| `-переменная->` | Делает переменную неинициализированной (drop) | `-переменная->` |
| `mv->` | Работает аналогично `->`, но также дропает значение | `переменная mv-> функция()` |
| `!->` | Аналог `.expect("текст")`, но в виде стрелки | `функция() !-> "Текст ошибки"` |
| `->\|` | Аналог `break` в Rust | `while true { println!("Текст"); ->\| }` |
| `->^` | Аналог `continue` в Rust | `while true { println!("Hello!"); if true { ->^ } }` |
| `->()` | Возвращает пустой кортеж из функции | `->()` |
| `&->` | Работает как `->`, но передаёт ссылку | `переменная &-> функция()` |  

## Новые ключевые слова
- `pass` – ничего не делает,используется для маркировки намерно пустых веток кода
- `TODO` - ничего не делает,используется для маркировки незаконченого кода
- `import std;` – импортирует базовые возможности из стандартной библиотеки Rust.
- `tuple имя (кортеж)` – обёртка над кортежами-структурами.
- `customtype имя тип` – обёртка над `type` из Rust.
- `var` - Встроеный макрос который заменяется на `let mut`

## Циклы
ArrowRust добавляет два новых цикла: `repeat` и `enumeration`.

### Repeat
```rust
repeat число as куда_деть_индекс { // куда_деть_индекс опционален, по умолчанию i
    // код
}
```
Вариант бесконечного цикла:

```rust
repeat {
    // код
}
```
### Enumeration
Встроенный цикл для перебора типов, которые поддерживают трейт Index и имеют метод .len().

```rust
enumeration from переменная1 to переменная2 {
    // код
}
```
### Стандартная библиотека (STD)
ArrowRust предоставляет набор готовых решений для простейших задач:

std/Base.rs – для работы с безопасными типами и динамическим типом Value.

std/Random.rs – для генерации случайных символов и чисел.

std/Bits.rs – для побитовых операций (установка конкретного бита, чтение бита).

std/Terminal.rs – для работы с цветом и терминалом.

# Новые типы:

 - SafeString – для хранения строк как вектора символов char, а не набора байт.

 - SafeVector – динамический массив без строгой типизации, автоматически расширяющийся.

 - Value – тип для хранения других типов.

 - SafeString и SafeVector обладают обычным функционалом строки и вектора соответственно.

 - Value может содержать:

 - целые числа: i8–i128, u8–u128

 - числа с плавающей точкой: f32, f64

 - Vec<u8>

 - String

 - char

 - bool

 - None

Value::None – это символ (char), созданный для обработки ошибок и затирания данных нулями. Рекомендуется использовать следующие пять базовых значений:

'\0' – затирание байтов и возврат «ничего».

'r' – ошибка чтения.

'b' – битые данные.

's' – успешно, но возвращать нечего.

Остальные значения свободны для пользовательских кодов.
